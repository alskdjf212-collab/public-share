Cowork란 무엇인가?

"이 엑셀 파일 정리해줘", "다운로드 폴더 분류해줘", "이 자료들로 보고서 만들어줘"—이런 요청을 AI에게 직접 맡기고, 실제로 파일이 완성되어 나온다면 어떨까요?

Claude Cowork는 Anthropic이 2026년 1월에 출시한 새로운 기능입니다. 개발자를 위한 Claude Code의 에이전트 아키텍처를 비개발자도 쓸 수 있게 만든 도구로, Claude Desktop 앱에서 터미널 없이 사용할 수 있습니다.

일반적인 AI 채팅과 가장 큰 차이점은 실행 능력입니다. 일반 채팅은 텍스트로 답변만 제공하지만, Cowork는 여러분의 컴퓨터 파일을 직접 읽고, 수정하고, 새로운 파일을 만들어냅니다. Anthropic은 이 경험을 "동료에게 메모를 남기는 것"에 비유합니다.

💡 핵심 차이: 채팅 vs Cowork

구분 일반 채팅 Cowork
파일 접근 수동 업로드 필요 로컬 폴더 직접 접근
작업 방식 질문-답변 1회 다단계 자율 실행
결과물 텍스트 응답 실제 파일 생성 (docx, xlsx, pptx, html 등)
작업 시간 즉시 응답 장시간 복잡한 작업 가능
도구 연동 제한적 Slack, Calendar, Gmail, Chrome 등

3단계로 시작하기

Cowork를 시작하는 데 5분이면 충분합니다. 복잡한 설정 없이, 앱을 열고 폴더를 지정하면 바로 사용할 수 있습니다.

1

Claude Desktop 설치

claude.ai/download에서 macOS 또는 Windows 버전을 다운로드합니다. 설치 후 Anthropic 계정으로 로그인하세요. Pro($20/월) 이상 유료 플랜이 필요합니다.

2

Cowork 모드 전환

앱 왼쪽 사이드바에서 "Cowork" 탭을 클릭합니다. 새로운 Cowork 세션이 시작되며, 작업할 폴더를 선택할 수 있는 화면이 나타납니다.

3

폴더 선택 & 작업 지시

Claude가 접근할 폴더를 선택하고, 자연어로 원하는 작업을 설명합니다. Claude는 계획을 수립하고, 단계별로 실행하며, 필요시 확인을 요청합니다.

⚠️ 시작 전 체크리스트

  • Claude Desktop 앱이 최신 버전인지 확인하세요
  • Pro, Max, Team, Enterprise 중 하나의 유료 플랜이 활성화되어 있어야 합니다
  • 작업 중에는 앱을 열어두어야 합니다 — 앱을 닫으면 세션이 종료됩니다
  • 인터넷 연결이 필요합니다

핵심 기능 5가지

Cowork가 단순한 챗봇이 아닌 진정한 업무 파트너가 되는 이유, 5가지 핵심 기능을 살펴보겠습니다.

1

로컬 파일 직접 접근

파일을 일일이 업로드하고 다운로드할 필요가 없습니다. 지정한 폴더 안의 파일을 Claude가 직접 읽고, 수정하고, 새 파일을 만들어냅니다. 엑셀, 워드, 파워포인트, PDF 등 다양한 형식을 지원합니다.

파일 관리 자동화
2

서브에이전트 병렬 처리

복잡한 작업을 여러 개의 소작업으로 나눈 뒤, 서브에이전트들이 동시에 처리합니다. 예를 들어 리서치와 문서 작성을 병렬로 진행해서 시간을 크게 단축합니다.

서브에이전트 병렬 실행
3

전문 문서 생성 (스킬)

수식이 살아있는 Excel 스프레드시트, 디자인된 PowerPoint 프레젠테이션, 서식이 적용된 Word 문서까지. 스킬(Skills) 시스템이 전문가 수준의 결과물을 만들어냅니다.

docx xlsx pptx pdf
4

외부 서비스 연동 (커넥터)

Slack 메시지 검색, Google Calendar 일정 확인, Gmail 읽기, Chrome 브라우저 자동화까지. 커넥터를 통해 다양한 외부 서비스와 연동하여 작업 범위를 확장합니다.

Slack Calendar Chrome
5

장시간 자율 작업

일반 채팅처럼 대화가 끊기지 않습니다. 복잡한 다단계 작업을 장시간에 걸쳐 자율적으로 수행하며, 중간에 방향을 수정하거나 추가 지시를 내릴 수 있습니다.

에이전트 자율 실행

서브에이전트 활용법

Cowork의 가장 강력한 기능 중 하나가 서브에이전트(Sub-agent) 시스템입니다. Claude가 복잡한 작업을 받으면, 이를 전문화된 소작업으로 분할하고 각각에 맞는 에이전트를 배치합니다.

에이전트 전문 영역 사용 도구
Bash 커맨드 실행, git 작업, 터미널 명령 Bash
general-purpose 복잡한 질문 조사, 코드 검색, 다단계 작업 모든 도구
Explore 파일 패턴 검색, 키워드 검색, 구조 파악 읽기 전용 도구
Plan 구현 전략 설계, 단계별 계획, 아키텍처 검토 읽기 전용 도구
claude-code-guide Claude Code/Agent SDK/API 관련 질문 검색 & 웹 도구

서브에이전트는 어떻게 작동하나요?

사용자가 직접 에이전트를 호출할 필요가 없습니다. 자연어로 요청만 하면 Claude가 자동으로 적합한 에이전트를 선택하고 실행합니다. 필요한 경우 여러 에이전트를 병렬로 동시에 실행하기도 합니다.

사용자: "이 프로젝트 폴더 분석하고, 구현 계획 세워줘" Claude의 내부 동작: → Explore 에이전트 → 폴더 구조 & 코드 탐색 → Plan 에이전트 → 분석 결과 기반 구현 계획 수립 → 두 에이전트가 병렬로 실행됨
사용자: "npm 빌드 실행하고 에러 있으면 고쳐줘" Claude의 내부 동작: → Bash 에이전트 → npm 명령어 실행 → general-purpose 에이전트 → 에러 분석 & 수정
명시적으로 지정하고 싶다면? "Explore 에이전트 써서 코드베이스 검색해줘"처럼 에이전트 이름을 직접 언급하면 해당 에이전트를 우선 사용합니다.

스킬(Skills)로 전문 문서 만들기

스킬은 특정 파일 형식이나 작업에 특화된 전문 모듈입니다. 각 스킬에는 수많은 시행착오를 거쳐 최적화된 베스트 프랙티스가 내장되어 있어, 전문가 수준의 결과물을 만들어냅니다.

📄

docx

Word 문서 생성/편집/분석. 서식, 목차, 머리글/바닥글까지

📊

xlsx

Excel 스프레드시트. 수식, 차트, 데이터 분석 포함

🎬

pptx

PowerPoint 프레젠테이션. 레이아웃, 디자인 적용

📑

pdf

PDF 생성, 병합, 분할, 텍스트 추출, 양식 작성

⚙️

skill-creator

나만의 커스텀 스킬 제작 및 성능 평가

⚡

create-shortcut

반복 작업 자동화를 위한 단축키 생성

스킬은 어떻게 사용하나요?

스킬도 서브에이전트와 마찬가지로 자연어로 요청하면 자동으로 호출됩니다. "보고서 Word 문서로 만들어줘"라고 말하면 docx 스킬이 자동으로 활성화되어, 전문적인 서식이 적용된 문서가 만들어집니다.

"이번 달 매출 데이터를 분석해서 Excel 스프레드시트로 정리해줘. 차트도 포함하고, 전월 대비 변화율도 계산해줘." → xlsx 스킬 자동 호출 → 수식, 차트, 서식이 포함된 .xlsx 파일 생성
"이 자료들을 종합해서 20페이지 분량의 프레젠테이션 만들어줘. 임원 보고용으로 깔끔한 디자인 부탁해." → pptx 스킬 자동 호출 → 레이아웃, 디자인이 적용된 .pptx 파일 생성

커넥터 & 플러그인

Cowork는 파일 작업만 하는 것이 아닙니다. 커넥터(Connector)를 통해 외부 서비스와 연동하고, 플러그인(Plugin)으로 기능을 확장할 수 있습니다.

기본 제공 커넥터

커넥터 할 수 있는 작업
Slack 채널/메시지 검색, 메시지 전송, 캔버스 생성, 사용자 조회
Google Calendar 일정 조회, 빈 시간 찾기, 캘린더 목록 확인
Gmail 이메일 검색, 메시지 읽기, 스레드 읽기
Figma 디자인 컨텍스트 추출, 스크린샷, 코드 생성
Chrome 브라우저 웹 페이지 탐색, 폼 작성, 스크린샷, 데이터 추출

플러그인 시스템

플러그인은 스킬, 커넥터, 슬래시 명령어, 서브에이전트를 하나의 번들로 통합한 확장 패키지입니다. 생산성, 영업, 재무, 법무, 마케팅 등 다양한 카테고리의 플러그인이 제공됩니다.

✅ 활용 팁: 커넥터 조합

"오늘 캘린더에 있는 미팅 내용을 확인하고, 관련 Slack 채널의 최근 논의를 정리해서, 미팅 준비 자료를 Word 문서로 만들어줘"—이런 식으로 여러 커넥터와 스킬을 조합하면, 하나의 요청으로 복합적인 작업을 완료할 수 있습니다.

실전 활용 시나리오 7가지

이론은 충분합니다. 실제로 Cowork를 업무에서 어떻게 활용할 수 있는지, 구체적인 시나리오와 프롬프트 예시를 살펴보겠습니다.

1

다운로드 폴더 자동 정리

수백 개의 파일이 뒤섞인 다운로드 폴더를 카테고리별(문서, 이미지, 동영상, 압축파일 등)로 자동 분류하고, 날짜별 하위 폴더를 만들어 정리합니다.

"다운로드 폴더를 정리해줘. 파일 종류별로 폴더를 만들고, 이름 규칙도 적용해줘. 중복 파일은 알려줘."
2

영수증에서 지출 보고서 작성

영수증 이미지들을 분석하여 날짜, 금액, 카테고리를 추출하고, 자동으로 Excel 지출 보고서를 생성합니다. 차트와 요약 통계도 포함됩니다.

"receipts 폴더에 있는 영수증 이미지들을 분석해서 이번 달 지출 보고서를 엑셀로 만들어줘. 카테고리별 합계와 차트도 포함해줘."
3

산재된 노트를 보고서로 통합

여러 파일에 흩어져 있는 회의록, 메모, 참고자료를 분석하여 체계적인 보고서 한 권으로 통합합니다.

"notes 폴더에 있는 회의록들을 읽고, 프로젝트 진행 보고서를 Word 문서로 만들어줘. 일정, 이슈, 다음 단계를 포함해줘."
4

시장 분석 프레젠테이션

웹 검색으로 시장 데이터를 수집하고, 분석 결과를 바탕으로 임원 보고용 프레젠테이션을 자동 생성합니다.

"2026년 AI SaaS 시장 동향을 조사해서 투자자 미팅용 프레젠테이션을 15슬라이드로 만들어줘."
5

일일 업무 브리핑 자동 생성

Slack, Calendar, Gmail의 정보를 종합하여 오늘의 일정, 중요 메시지, 대응이 필요한 이메일을 한 눈에 볼 수 있는 브리핑 문서를 만듭니다.

"오늘 캘린더 일정 확인하고, Slack에서 나에게 온 중요 메시지 정리하고, Gmail 미읽은 메일 요약해서 오늘의 브리핑 문서 만들어줘."
6

블로그 글 작성 (HTML)

주제를 주면 웹 리서치, SEO 최적화, 본문 작성, 메타 태그까지 포함된 완성도 높은 HTML 블로그 글을 생성합니다.

"prompt.md 가이드를 참고해서 '재택근무 생산성 높이는 방법' 주제로 SEO 최적화된 HTML 블로그 글을 작성해줘."
7

데이터 분석 & 시각화

CSV나 Excel 데이터를 분석하여 트렌드를 파악하고, 인사이트가 담긴 차트와 대시보드를 생성합니다.

"sales_data.csv 파일을 분석해줘. 월별 매출 추이, 상위 10개 제품, 지역별 분포를 차트로 시각화하고 인사이트를 정리해줘."

보안 & 주의사항

AI에게 파일 접근 권한을 주는 것이 걱정될 수 있습니다. Cowork는 여러 겹의 안전장치를 갖추고 있습니다.

보안 아키텍처

🔒 격리된 가상 머신 (VM)

Cowork는 컴퓨터 안에서 격리된 가상 머신(VM)으로 실행됩니다. Apple Virtualization Framework를 사용한 커스텀 Linux 환경에서 작업하므로, Claude가 시스템 전체에 접근하는 것이 아닙니다.

📁 명시적 폴더 접근 권한

사용자가 직접 선택한 폴더만 Claude가 접근할 수 있습니다. 그 외의 파일이나 시스템 영역에는 접근할 수 없습니다.

🛡️ 파일 삭제 보호

파일을 영구 삭제하기 전에 반드시 사용자의 명시적 허가를 요청합니다. 허가 프롬프트가 표시되며, "Allow"를 클릭해야 삭제가 진행됩니다.

💾 로컬 데이터 저장

Cowork 대화 기록은 Anthropic 서버가 아닌 로컬 장치에 저장됩니다.

⚠️ 주의해야 할 점

  • 중요한 파일은 백업하세요 — AI가 의도치 않게 파일을 수정할 가능성이 있습니다
  • 민감한 정보(비밀번호, 금융 정보 등)가 포함된 폴더에는 접근 권한을 부여하지 마세요
  • 규제 대상 데이터(의료, 법률 등)에는 Cowork를 사용하지 마세요
  • 결과물은 반드시 사실 확인(Fact-check)을 거치세요

자주 묻는 질문 (FAQ)

Q: Cowork는 무료로 사용할 수 있나요?

Cowork는 Claude Pro($20/월), Max($100~$200/월), Team, Enterprise 등 유료 플랜 구독자에게 리서치 프리뷰로 제공됩니다. Pro 사용자도 모든 기능에 접근할 수 있지만, 복잡한 작업은 더 많은 사용량을 소비하므로 한도에 더 일찍 도달할 수 있습니다.

Q: Windows에서도 사용할 수 있나요?

2026년 2월 10일부터 Windows(x64)에서도 Claude Desktop을 통해 Cowork를 사용할 수 있습니다. macOS와 동일한 기능을 지원합니다.

Q: 내 파일이 Anthropic 서버로 전송되나요?

Cowork는 컴퓨터의 격리된 가상 머신에서 로컬로 실행됩니다. 대화 기록은 로컬 장치에 저장되며, 사용자가 명시적으로 접근 권한을 부여한 폴더만 Claude가 접근할 수 있습니다.

Q: 일반 채팅과 Cowork를 동시에 사용할 수 있나요?

네, 가능합니다. Claude Desktop 앱에서 일반 채팅 탭과 Cowork 탭을 각각 별도로 열어 사용할 수 있습니다. 두 모드는 독립적으로 작동합니다.

Q: 커넥터(Slack, Gmail 등)는 어떻게 연결하나요?

Cowork 세션에서 해당 서비스가 필요한 작업을 요청하면, Claude가 자동으로 커넥터 연결을 안내합니다. 또는 Cowork 설정에서 직접 커넥터를 추가할 수도 있습니다. OAuth 인증을 통해 안전하게 연결됩니다.
